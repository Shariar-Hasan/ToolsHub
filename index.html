T-Shirt Size Detection

It is an innovative web app that helps you find your perfect t-shirt size using the power of Machine Learning. It utilizes the K-Nearest Neighbors (KNN) algorithm to predict your ideal t-shirt size based on your height and width measurements. With it, you can say goodbye to ill-fitting t-shirts and confidently order your preferred size online.

Word Operation Count

This web app which have UI consists of two input fields where the user can enter the two words they want to compare. After entering the words, they can click the "Calculate" button, and the app will display the edit distance value and the sequence of operations needed to transform the first word into the second word.
